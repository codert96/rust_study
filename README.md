# axum-di-router AI随便写的readme

一个基于 Rust Axum 框架构建的轻量级项目，核心实现了自定义的 Bean 注入（依赖注入）和自动化路由收集功能，旨在简化 Axum 项目的开发流程，提升代码的模块化和可维护性。

## 项目介绍

该项目聚焦于 Axum 框架的基础能力增强，目前暂未实现具体业务功能，核心价值在于：
- 自定义 Bean 注入机制：实现轻量级的依赖注入，解耦组件依赖
- 自动化路由收集：无需手动注册路由，通过注解/宏自动扫描并注册路由
- 遵循 Rust 简洁、高性能的设计理念，与 Axum 原生特性深度兼容

## 技术栈
- 编程语言：Rust (≥ 1.70)
- Web 框架：Axum (≥ 0.7)
- 核心特性：自定义依赖注入、自动化路由收集
